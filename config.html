When a message has some safe signals but not enough to pass, Claude reads the subject + first 1,500 chars of the body and decides: is this a real person or automated mail? Costs roughly $0.0001 per grey-zone email. Requires an Anthropic API key.
You’ll need an API key from console.anthropic.com. After installing the script, open Apps Script → Project Settings → Script Properties and add a property called ANTHROPIC_API_KEY with your key as the value. Then run testAIClassifier to confirm it works.
